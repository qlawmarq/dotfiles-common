# Snowflake 進行表

> **モード: 構造化** — Ingermanson の Snowflake Method。1 文 → 1 段落 → 1 ページとフラクタル拡張。各段階の終わりで「次に進めるか／戻るか」のみ判定。戻る場合はそのセッション中に必ず次段階に戻る。

## ステップ 1: 1 文ログライン

形式: 「**[主人公] が [障害] にもかかわらず [目的] を追う物語**」

ゲームの場合はコア動詞を含める。

### 候補（3-5 個）

1.
2.
3.

### 採用版（25-50 字以内）

> {{ここに最終版を書く}}

---

| 3. 危機 2 | |
| 4. 危機 3 | |

## 進行判定

各段階終了時に判定:

- [ ] ステップ 1 → 2 に進む（戻らないなら ✓）
- [ ] ステップ 2 → 4 に進む
- [ ] ステップ 4 → 執筆フェーズに進む

戻る場合はそのセッション中に必ず次段階に戻ること（戻り続けると Snowflake 地獄に陥る）。
